# Exercise Express

You've been tasked with creating a simple API using the framework Express. Tailwind Traders woulld like to evaluate Express and see if it's easy to work with. As part of that evaluation they would like you to build a simple web application that server different routes.

# Create a basic Web app with Express

You will create a basic application that will be able to handle requests.

  1. Create a file app.js.

  2. Go to the terminal and enter npm init -y. This will initialize a Node.js project.

  3. In the terminal type npm install express. This will install the Express framework.

  4. Open up the file package.json and note how the dependencies section now has an express entry. That means Express has been correctly installed.

  5. Add the following code to app.js:

    const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => res.send('Hello World!'))
app.listen(port, () => console.log(`Example app listening on port ${port}!`))

    node app.js

    This means your app is up and running and ready to receive requests.

  7. Open a up a browser and navigate to http://localhost:3000. In the browser window you should now see the following text displayed Hello World!.

  8. Type Ctrl-C in the terminal to stop the program.
